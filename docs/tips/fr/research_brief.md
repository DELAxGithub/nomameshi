# Nomameshi — フランス Cultural Tips リサーチ＆拡充 発注書

## 1. 概要

海外メニュー翻訳アプリ「Nomameshi」のフランス（FR）向け Cultural Tips を拡充するためのリサーチ発注書です。現在3件（日本語・英語 各3件）あり、**43件**を目標に追加リサーチを依頼します。

## 3. 追加リサーチの方向性

以下のカテゴリで **各5〜8件** を目安に、現在カバーされていない新規 Tips をリサーチしてください。

### A. 地方料理・食材（優先度：高）
- **リヨン**: ブション（bouchon）文化、クネル、サラダ・リヨネーズ、ローヌ=アルプのチーズ
- **アルザス**: シュークルート、タルト・フランベ（フラムクーヘン）、ブレッツェル、リースリングとの合わせ方
- **プロヴァンス**: ラタトゥイユ、ブイヤベース（マルセイユ）、アイオリ、ニース風サラダの正統派
- **ブルターニュ**: ガレット（そば粉クレープ）vsクレープ（小麦粉）の違い、シードル、オイスター
- **ノルマンディー**: カマンベール、カルヴァドス、トルー・ノルマン（コースの合間の一杯）
- **バスク**: ピペラード、アシュア、エスペレット唐辛子、ガトー・バスク
- **ボルドー**: カヌレ、アントルコート・ボルドレーズ、ワインとのペアリング文化
- **南西部（ペリゴール・トゥールーズ）**: フォアグラ、カスレ、鴨のコンフィ
- 季節限定メニュー（ジビエ、白アスパラガス、ボジョレー・ヌーヴォーなど）

### B. ワイン＆ドリンク（優先度：高）
- 食前酒（アペリティフ）文化 — キール、パスティス、リレ
- ワインの基本: ボルドー/ブルゴーニュ/ローヌ/アルザス/ロワールの特徴
- レストランでのワイン注文: ピシェ（carafe）vsボトル、ヴァン・ド・ラ・メゾン
- カフェでの注文: un café = エスプレッソ、café crème/noisette/allongé の違い
- 食後酒（ディジェスティフ）: カルヴァドス、コニャック、アルマニャック
- ノンアルコール: シロ（sirop）+ 水、ディアボロ（sirop + limonade）

### C. パン・スイーツ（優先度：中）
- ブーランジュリー（パン屋）vs パティスリー（菓子屋）の違い
- 朝はクロワッサン/パン・オ・ショコラが定番（ヴィエノワズリー）
- パンの種類: バゲット・トラディション（伝統製法）が美味しい理由
- パティスリーの定番: タルト・タタン、ミルフィーユ、エクレア、パリ＝ブレスト
- チョコラティエ文化とバレンタインの違い

### D. マナー・文化（優先度：中）
- 店に入ったら必ず「Bonjour」、出る時「Au revoir」— これを省くと冷たい対応になる
- フォーマルディナーのコース順序: アペリティフ → アントレ → プラ → フロマージュ → デセール → カフェ
- フロマージュ（チーズ）は食後・デザート前に出る独自文化
- チーズの切り方ルール（形によって切り方が違う）
- 食事中の手の位置（テーブルの上に両手を見せる）
- パンは皿の左上、直接テーブルに置くのもOK（パン皿がない店も多い）

### E. 実用・注文テクニック（優先度：中）
- menu（コース）vs carte（アラカルト）vs formule（ランチセット）の違い
- plat du jour（本日の一品）はコスパ◎、黒板に手書きが多い
- アレルギー・食事制限の伝え方（グルテンフリー事情はまだ発展途上）
- レストラン予約の重要性（特に週末・人気店）
- 営業時間: ランチ12〜14時、ディナー19〜21時30分が一般的、間の時間は閉まる
- カフェ vs ビストロ vs ブラッスリー vs レストランの格の違い
- 会計は「L'addition, s'il vous plaît」、テーブルで支払い

### F. 市場・買い物（優先度：中）
- マルシェ（市場）は週1〜2回開催の町が多い、午前中が勝負
- フロマージュリー（チーズ専門店）での買い方 — 「pour aujourd'hui」と言えば食べ頃を選んでくれる
- AOC/AOP（原産地統制呼称）の意味と信頼性

### G. フランス語フレーズ（優先度：低〜中）
- 現在「service compris」をカバー済み
- 追加候補:
  - 注文: 「Je voudrais...（ジュ・ヴドレ）」「L'addition（ラディシオン）」
  - 感想: 「C'est délicieux!（セ・デリシュー）」「C'était très bon（セテ・トレ・ボン）」
  - 実用: 「une carafe d'eau（ユヌ・カラフ・ドー）」で無料の水道水
  - おすすめを聞く: 「Qu'est-ce que vous conseillez?（ケスク・ヴ・コンセイエ）」

### H. カフェ文化（優先度：中）
- テラス席 vs 店内で値段が違う（パリ中心部は特に）
- カフェでの長居は基本OK、ただしランチタイムは回転重視
- 朝のカフェ＋クロワッサン（petit déjeuner）の文化

### I. コスパ・節約（優先度：低〜中）
- ランチの formule/menu は €15〜25 でフルコース並み
- ブーランジュリーのサンドイッチ（ジャンボン・ブール等）は €4〜6
- スーパーの惣菜（traiteur）活用

### J. ノマド・リモートワーク（優先度：低）
- パリのコワーキングカフェ事情
- WiFi のあるカフェの見つけ方、長居の相場感

## 4. Tips のトーン＆スタイルガイド

### 基本ルール
- **1 Tip = 1知識**。複数トピックを詰め込まない
- **先頭に 💡 を付ける**（アプリ内表示のプレフィックス）
- **「知ってる人」が友達に教えるトーン**。ガイドブック的な堅さはNG
- **具体的な数字・固有名詞を入れる**（€の相場、地名、料理名）
- **行動に繋がる情報を優先**（「〜するのがおすすめ」「〜と言えばOK」）
- **フランス語のフレーズを含む場合はカタカナ読みも添える**（日本語版）

### 文字数
- **日本語**: 80〜180文字（目安）。スマホ画面で2〜4行に収まる長さ
- **英語**: 同等の情報量。自然な英語で、直訳しない

### NGパターン
- ❌ 「フランスでは〜です。」だけの教科書的記述
- ❌ Wikipedia的な歴史解説（由来は1行ならOK）
- ❌ 他のTipと内容が重複する情報
- ❌ 特定の店名やブランド名の推薦（一般論で書く）
- ❌ ネガティブな表現（「まずい」「避けろ」）→ ポジティブに言い換え

### OKパターン（参考例）
- ✅ 「リヨンの『ブション』は地元の家庭料理を出す小さな食堂。クネル（白身魚のすり身団子）を頼めば店の実力がわかる。Bouchon認定マーク付きの店が安心。」
- ✅ 「フランスでは食後にチーズが出てくる。デザートの前。ワインが残っていればチーズと合わせるのが定番の流れ。」
- ✅ 「カフェで『un café（アン・カフェ）』と頼むとエスプレッソが出てくる。日本式のコーヒーが欲しければ『un café allongé（アロンジェ）』と言おう。」

## 5. 納品形式

### ファイル形式
JSON ファイルで納品してください。以下の構造に従ってください：

```json
{
  "region": "FR",
  "version": "1.0",
  "date": "2026-MM-DD",
  "tips": [
    {
      "id": "fr_004",
      "category": "地方料理",
      "ja": "💡リヨンの『ブション』は...",
      "en": "💡Lyon's \"bouchon\" restaurants are...",
      "tags": ["lyon", "bouchon", "regional"],
      "source_notes": "リサーチ元のメモ（納品後削除可）"
    }
  ]
}
```

### フィールド説明
| フィールド | 必須 | 説明 |
|-----------|------|------|
| `id` | ✅ | `fr_` + 3桁連番。既存は fr_001〜fr_003 |
| `category` | ✅ | 下記のカテゴリから選択 |
| `ja` | ✅ | 日本語テキスト（💡プレフィックス付き） |
| `en` | ✅ | 英語テキスト（💡プレフィックス付き） |
| `tags` | ✅ | 検索・フィルタ用タグ（3〜5個、英語小文字） |
| `source_notes` | ❌ | リサーチソースのメモ（最終実装時は除外） |

### カテゴリ一覧
- `食事時間` — いつ何を食べるか
- `料理知識` — 料理・食材の知識
- `地方料理` — 地方別の特徴・名物
- `地方文化` — 地方固有の食文化・習慣
- `ドリンク` — ワイン・カフェ・飲み物全般
- `マナー` — 食事マナー・エチケット
- `フランス語` — 便利なフレーズ
- `実用` — 価格・営業・予約等の実用情報
- `コスパ` — 節約・コスパ情報
- `カフェ文化` — カフェ特有の文化・ルール
- `ノマド` — リモートワーカー向け情報
- `市場・買い物` — マルシェ・食材ショッピング
- `パン・スイーツ` — ブーランジュリー・パティスリー文化

### 品質チェックリスト
納品前に以下を確認してください：

- [ ] 既存3件との重複なし
- [ ] 各 Tip が1トピックに絞られている
- [ ] 日本語 80〜180文字の範囲内
- [ ] 英語版が直訳でなく自然な英文
- [ ] 💡プレフィックスが付いている
- [ ] フランス語フレーズにカタカナ読みあり（日本語版）
- [ ] 具体的な数字・地名が含まれている
- [ ] tags が3〜5個付いている
