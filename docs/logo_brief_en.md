# Nomameshi - Logo Design Brief (v2)

## 1. About This Brief
This is a **logo-only** redesign brief. The app's UI rebranding (color palette, typography, screen layouts) has already been completed. We now need a logo that matches the new warm, approachable direction.

## 2. Project Context
**Nomameshi** is a menu translation app for global nomads. Users photograph a foreign-language restaurant menu, and the app translates, explains, and visualizes the dishes using AI.

- **Name origin**: Nomad + Meshi (everyday meal in Japanese). Inspired by NHK's "Salameshi" (salaryman's lunch).
- **Tagline concept** (optional, for logo lockup): "Your local menu, translated."
- **Live URL**: https://menumenu-three.vercel.app

## 3. Target Audience
- Slow nomads who settle in one place for weeks/months
- Adventurous foodies who prefer local neighborhood eateries over tourist spots
- Working professionals having everyday meals abroad, not vacation dining

## 4. Brand Personality
| Attribute | Description |
|---|---|
| **Warm** | Like a handwritten menu at a local cafe |
| **Approachable** | Friendly, not techy or corporate |
| **Authentic** | Real local dining, not curated Instagram food |
| **Practical** | A daily-use utility, not a luxury experience |
| **Global** | Works across cultures without being tied to one |

**What to avoid**: Cold/tech/cyber feel. Resort or vacation vibes. Overly cute or illustrative.

## 5. Current Color Palette (Implemented)
The logo must work harmoniously with these colors:

| Token | Value | Usage |
|---|---|---|
| `--background` | `#FCFBF9` | Page background (ivory) |
| `--foreground` | `#1E2432` | Primary text (charcoal) |
| `--primary` | `#E85A4F` | CTA / Accent (coral) |
| `--secondary` | `#8A9178` | Emphasis (olive) |
| `--accent` | `#C9A86C` | Section titles (soy/gold) |

## 6. Current Typography
- **Headings**: Outfit (Google Font), weight 700
- **Body**: Inter (Google Font)
- The logo may use a custom/different typeface for character, but it should feel cohesive with these fonts.

## 7. Logo Requirements

### 7-1. Type
- **Typography-focused wordmark** — the app name "nomameshi" as the primary mark
- An optional simple icon/symbol is welcome (but not required) for favicon/app-icon use
- No elaborate illustrations or mascots

### 7-2. Style Direction
- Lowercase preferred ("nomameshi"), approachable and casual
- Consider subtle warmth cues: slight roundness, a gentle ligature, or a soft serif/humanist quality
- The Japanese concept of "meshi" (everyday meal) can optionally be hinted at — e.g., through chopstick-inspired strokes, a subtle rice-bowl reference, or steam — but keep it minimal and tasteful
- Should feel like it could live on a cafe menu board or a nomad's laptop sticker

### 7-3. Color Variations Needed
1. **Primary**: Charcoal (`#1E2432`) on ivory background (`#FCFBF9`)
2. **Reversed**: Ivory/white on dark or photo backgrounds (for result-mode overlay)
3. **Monochrome**: Single-color black and white versions
4. **Accent**: Coral (`#E85A4F`) version for special contexts

### 7-4. Size & Usage Contexts
| Context | Size/Format | Notes |
|---|---|---|
| App header | ~24px height | Displayed at top of mobile screen |
| Favicon | 32x32, 16x16 | Icon-only version needed |
| PWA app icon | 192x192, 512x512 | Square with rounded corners |
| OG image | 1200x630 | Social share preview |
| Menu card footer | ~12px height | "Translated by nomameshi" credit line |
| Sticker/merch | Print-ready | Vector format required |

### 7-5. File Formats
- SVG (vector, primary)
- PNG with transparency (various sizes)
- Favicon formats (ICO, PNG)

## 8. Reference & Inspiration
For tone/mood reference, think of brands like:
- **Notion** — clean wordmark, minimal, approachable
- **Are.na** — understated, community-focused
- **Muji** — simple, warm, everyday utility
- **A neighborhood cafe's hand-lettered sign** — warm but not sloppy

Currently using plain text "nomameshi" in the Outfit font. This works as a placeholder but lacks distinct character and brand recognition.
