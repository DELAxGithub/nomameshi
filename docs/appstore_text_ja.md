## 説明 (4000文字以内)
```
外国語メニューを撮影するだけ。Nomameshiが瞬時に翻訳し、AIが料理のビジュアルを生成。注文前に「味を見る」新体験を。

■ できること
・レストランのメニューを撮影またはアップロード
・Gemini AIによる自然な翻訳をリアルタイム表示
・各料理のAI生成イメージで、何を頼むか一目瞭然
・メニュー全体を再現した「テーブルスプレッド」画像を自動生成

■ 対応言語
あらゆる言語のメニューを日本語・英語に翻訳。日本・イタリア・スペイン・フランス・アメリカ・韓国・タイ・台湾のメニューに最適化。

■ スマート機能
・メニューの国・言語を自動検出
・セクション別に料理を整理、価格も表示
・馴染みのない料理には簡潔な文化的解説つき
・待ち時間に楽しい食文化トリビアを表示

■ 共有・保存
・翻訳テキストをクリップボードにコピー
・お気に入りのアプリで共有
・翻訳結果を美しい画像として保存

旅行者、グルメ好き、海外で自信を持って注文したいすべての人に。アカウント登録不要 — 撮って、翻訳して、発見しよう。
```
