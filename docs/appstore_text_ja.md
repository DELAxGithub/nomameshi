## 説明 (4000文字以内)
```
観光客向けの店じゃなくて、地元の人が通うあの小さな食堂に入ってみたい。そう思ったときに隣にいてくれるアプリです。

外国語のメニューを撮影すると、AIが瞬時に翻訳し、その料理が実際どう見えるかのビジュアルまで自動で生成します。「英語メニューを用意してないお店」でも、安心して注文できる。それが Nomameshi の一番大事な役割です。

■ ローカル食堂での使い方
・気になる定食屋・バル・タベルナに入って、迷わず着席
・メニューを撮影 or アップロード → 料理ごとの翻訳とAIビジュアル
・「テーブルスプレッド」画像で全品一目瞭然、何を頼むか直感で決まる
・待ち時間はその国の食文化Tipsを読んで、料理を待つ時間さえ楽しくなる

■ なぜ Nomameshi なのか
・グーグル翻訳と違い「料理」に特化 — 地名・食材・調理法のニュアンスを大事にする
・単なる翻訳で終わらない — その料理が視覚的にどんなものかまで見える
・国別にキュレーションされたローカルTips（スペイン84件、フランス43件ほか拡充中）
・アカウント不要。撮って、翻訳して、食べる。それだけ。

■ 対応言語・地域
あらゆる言語のメニューを日本語・英語に翻訳。特に日本・イタリア・スペイン・フランス・アメリカ・韓国・タイ・台湾のローカル料理に最適化しています。

■ スマート機能
・メニューの国・言語を自動検出
・セクション別に料理を整理、価格も表示
・馴染みのない食材・料理にはさりげない文化的解説
・待ち時間のご当地Tipsは何度見ても飽きない量

■ 保存・シェア
・翻訳結果を美しい一枚の画像として保存
・お気に入りのアプリで友達にシェア
・旅の思い出として履歴に最大20件自動保存

スローノマド、食好きの旅人、「ローカルで食べる」を大事にしたいすべての人へ。Nomameshi はガイドブックでも翻訳機でもなく、旅先の食卓の相棒です。
```
