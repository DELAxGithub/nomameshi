## 3. 追加リサーチの方向性

以下のカテゴリで **各5〜10件** を目安に、現在カバーされていない新規 Tips をリサーチしてください。

### A. 地方料理・食材（優先度：高）
- **カナリア諸島**: モホソース、パパス・アルガーダス、gofio
- **ムルシア**: サラナゴージョ、パスティール・デ・カルネ
- **アラゴン**: テルナスコ、migas aragonesas
- **ナバラ**: 白アスパラガス、ピキージョ・ピーマン
- **カンタブリア**: ソバオ・パシエゴ、cocido montañés
- 各地方の「◯◯ a la ～」表現で未カバーのもの
- 季節限定メニュー（calçots, cochinillo en Navidad 等）

### B. メニューの読み方・注文テクニック（優先度：高）
- メニューのセクション構造（entrantes, primeros, segundos, postres）
- 「del día」「de temporada」「de mercado」の意味
- アレルギー/食事制限の伝え方
- ベジタリアン/ビーガンでの注文（伝統的に少ないが増加中）
- 「para compartir」（シェア用）と「ración」vs「media ración」
- 量の目安（1人何品が適切か）

### C. 市場・食材ショッピング（優先度：中）
- メルカード（市場）での買い物の基本
- オリーブオイルの等級・選び方
- チーズの地方別特徴（マンチェゴ以外）
- ワインの原産地呼称（D.O.）の基本

### D. 実用・マナー・文化（優先度：中）
- レストラン予約の文化（必要な場面と不要な場面）
- 食物アレルギー申告の法的義務（EU規制）
- サービス料（servicio incluido）の読み方
- 食事中のスマホマナー
- 子連れ食事の文化（スペインは子連れに寛容）
- 祝日・フェリア期間の営業事情

### E. スペイン語フレーズ（優先度：低〜中）
- 現在「¡Riquísimo!」「¡La cuenta!」「¡Oiga!」等をカバー済み
- 追加候補: 注文フレーズ、感想の伝え方、トラブル対応

## 4. Tips のトーン＆スタイルガイド

### 基本ルール
- **1 Tip = 1知識**。複数トピックを詰め込まない
- **先頭に 💡 を付ける**（アプリ内表示のプレフィックス）
- **「知ってる人」が友達に教えるトーン**。ガイドブック的な堅さはNG
- **具体的な数字・固有名詞を入れる**（€の相場、地名、料理名）
- **行動に繋がる情報を優先**（「〜するのがおすすめ」「〜と言えばOK」）
- **スペイン語のフレーズを含む場合はカタカナ読みも添える**

### 文字数
- **日本語**: 80〜180文字（目安）。スマホ画面で2〜4行に収まる長さ
- **英語**: 同等の情報量。自然な英語で、直訳しない

### NGパターン
- ❌ 「スペインでは〜です。」だけの教科書的記述
- ❌ Wikipedia的な歴史解説（由来は1行ならOK）
- ❌ 他のTipと内容が重複する情報
- ❌ 特定の店名やブランド名の推薦（一般論で書く）
- ❌ ネガティブな表現（「まずい」「避けろ」）→ ポジティブに言い換え

### OKパターン（既存Tipの良い例）
- ✅ 「クロケタスは店の実力が一番わかる一品。冷凍品は衣が厚く油っぽい。最初にこれを頼んで手作りか見極めるのが通の使い方。」
- ✅ 「ガンバス・アル・アヒージョを食べ終わったら残ったオイルにパンを浸すのが現地流。『¿Me pone más pan?』と言えばパーフェクト。」
- ✅ 「夏の『café con hielo』はホットエスプレッソと氷入りグラスが別々に出てくる。砂糖はホットのうちに溶かしてから氷に注ぐのがコツ。」

## 5. 納品形式

### ファイル形式
JSON ファイルで納品してください。以下の構造に従ってください：

```json
{
  "region": "ES",
  "version": "2.0",
  "date": "2026-MM-DD",
  "tips": [
    {
      "id": "es_045",
      "category": "地方料理",
      "ja": "💡カナリア諸島の「モホ・ピコン」は...",
      "en": "💡Canary Islands' \"mojo picón\" is...",
      "tags": ["canarias", "sauce", "regional"],
      "source_notes": "リサーチ元のメモ（納品後削除可）"
    }
  ]
}
```

### フィールド説明
| フィールド | 必須 | 説明 |
|-----------|------|------|
| `id` | ✅ | `es_` + 3桁連番。既存は es_001〜es_044 |
| `category` | ✅ | 下記のカテゴリから選択 |
| `ja` | ✅ | 日本語テキスト（💡プレフィックス付き） |
| `en` | ✅ | 英語テキスト（💡プレフィックス付き） |
| `tags` | ✅ | 検索・フィルタ用タグ（3〜5個、英語小文字） |
| `source_notes` | ❌ | リサーチソースのメモ（最終実装時は除外） |

### カテゴリ一覧
- `食事時間` — いつ何を食べるか
- `料理知識` — 料理・食材の知識
- `地方料理` — 地方別の特徴・名物
- `地方文化` — 地方固有の食文化・習慣
- `ドリンク` — 飲み物全般
- `マナー` — 食事マナー・エチケット
- `スペイン語` — 便利なフレーズ
- `実用` — 価格・営業・予約等の実用情報
- `コスパ` — 節約・コスパ情報
- `バル文化` — バル特有の文化・ルール
- `ノマド` — リモートワーカー向け情報
- `市場・買い物` — 市場・食材ショッピング（新規）
