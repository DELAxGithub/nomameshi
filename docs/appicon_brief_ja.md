# Nomameshi — アプリアイコン デザイン発注書

## 1. 概要

メニュー翻訳アプリ「Nomameshi」のiOS / Web 用**アプリアイコン**のデザインを依頼します。現在のアイコンはプレースホルダー（フレームワークのデフォルト）です。ホーム画面で映えるプロフェッショナルなアイコンが必要です。

**本発注書はアプリアイコン専用です。** ロゴワードマークは別途発注書があります — アイコンは将来のワードマークとファミリー感のあるデザインが望ましいですが、今回の納品物はアイコンのみです。

## 2. Nomameshi について

- **機能**: 外国語のレストランメニューを撮影 → AIが瞬時に翻訳＆料理ビジュアルを生成
- **名前の由来**: Nomad（遊牧民）+ Meshi（飯）。NHK「サラメシ」からインスピレーション
- **タグライン**: "See the Flavor"（味を見る）
- **ターゲット**: スローノマド、冒険好きなグルメ、海外で日常的に食事をするワーカー
- **公開中のアプリ**: https://nomameshi-delaxs-projects.vercel.app

## 3. ブランドパーソナリティ

| 属性 | 説明 |
|---|---|
| **温かみ** | 近所のカフェの手書きメニューのような |
| **親しみやすさ** | フレンドリーで、テック感やコーポレート感を出さない |
| **本物感** | インスタ映えの加工食ではなく、現地のリアルな食事 |
| **実用的** | 毎日使うツール、ラグジュアリー体験ではない |
| **グローバル** | 特定の文化に偏らず、世界中で通用する |

**避けたいもの**: 冷たいテック / サイバー感。リゾート / バカンスのイメージ。過度にかわいいマスコット。クリップアート風の食べ物イラスト。

## 4. カラーパレット（確定済み）

以下の既存パレットからアイコンの色を選んでください：

| 色名 | Hex | 役割 |
|---|---|---|
| アイボリー | `#FCFBF9` | 背景 |
| チャコール | `#1E2432` | 主テキスト / ダーク要素 |
| コーラル | `#E85A4F` | メインアクセント / CTA |
| オリーブ | `#8A9178` | セカンダリアクセント |
| ソイ / ゴールド | `#C9A86C` | 暖かいアクセント |

**推奨方向性**: コーラル (`#E85A4F`) をメインカラーまたは背景色に、チャコール / アイボリーでコントラストを出す。他のアプリの中で温かみがあり、すぐに見つけられるアイコンに。

## 5. コンセプト方向性（提案）

デザイナーの創造性にお任せしますが、以下の方向性を出発点としてご検討ください：

### 方向性A — 抽象的な「N」マーク
Nomameshiの「N」をスタイリッシュに。箸のストローク、湯気のカーブ、メニューページの折り目など、食 / 温かみのニュアンスをさりげなく込める。

### 方向性B — メニュー + レンズ
メニューやカード型にカメラ / レンズ要素を組み合わせ、「メニューを撮影する」というコアアクションを表現。ミニマル、ジオメトリック、モダン。

### 方向性C — 丼 / テーブルシンボル
料理、丼、テーブルセッティングを抽象化したシンボル。「メシ（飯）」を連想させる。リテラルなイラストではなく、洗練されたジオメトリック形状で食文化をほのめかす。

### 方向性D — タイポグラフィアイコン
「の」や「N」を大胆かつコンパクトにタイポ処理し、単独アイコンとして成立させる。29x29pxでも認識可能な強さが必要。

## 6. 技術仕様

### サイズ & フォーマット
| 用途 | サイズ | 備考 |
|---|---|---|
| App Store | 1024 x 1024 px | メイン納品物、PNG、透過なし |
| iOS ホーム画面 | 180 x 180 px (@3x) | 1024から自動生成 |
| Spotlight | 120 x 120 px (@3x) | 1024から自動生成 |
| 設定 | 87 x 87 px (@3x) | 1024から自動生成 |
| ファビコン | 32 x 32, 16 x 16 px | 必要に応じて簡略版 |
| Web (PWA) | 192 x 192, 512 x 512 px | manifest.json用 |

### Apple ガイドライン準拠
- **アルファチャンネル / 透過なし**（iOSアイコンは透過不可）
- iOSが自動的に角丸（スーパー楕円）を適用 — **アセットに角丸を含めないこと**
- 小サイズ（29pt / 87px）でも視認可能・認識可能であること
- 小サイズで消えてしまう細かいディテールは避ける
- アイコン内にテキストは入れない（Apple非推奨）
- フォーカスポイントは1つ、シンプルな構成

### 納品ファイル
1. **1024x1024 PNG**（透過なし）— App Store & iOS 用
2. **512x512 PNG** — PWA用
3. **192x192 PNG** — PWA用
4. **32x32 & 16x16 PNG** — ファビコン用
5. **SVG**（ベクター）— スケーラビリティ確保
6. **ソースファイル**（Figma / Illustrator / Sketch）

## 7. 表示コンテキスト

アイコンが表示される場面：
- **iPhoneホーム画面**: 多数のカラフルなアプリアイコンに囲まれる → 目立つ必要あり
- **App Store検索結果**: アプリ名「Nomameshi」の横に小さなサムネイル
- **Safari / ブラウザタブ**: 極小のファビコンとして
- **共有シート / 通知**: 様々な小サイズで表示

## 8. 競合アプリとの差別化

翻訳 / フード系の類似アプリ（これらと差別化する）：
- **Google翻訳**: 青白、カメラアイコン → コーポレートすぎる
- **Papago**: 緑のオウム → マスコット寄りすぎ
- **Yelp**: 赤い星 → 汎用的すぎる
- **Uber Eats**: 緑黒 → デリバリー寄り

目指す方向性に近いアプリ: **Airbnb**（温かみ、旅、親しみ）、**Notion**（クリーン、ミニマル）、**Bear**（暖色、シンプルアイコン）

## 9. 参考情報：現在のCSSワードマーク

アプリのヘッダーで使用中のCSSワードマーク：
- `noma` をチャコール (`#1E2432`) + `meshi` をコーラル (`#E85A4F`) で表示
- フォント: Outfit, weight 800, letter-spacing: -0.04em
- アイコンがこれを再現する必要はないが、同じファミリーに見える統一感は欲しい
