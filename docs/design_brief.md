# ノマメシ (Nomameshi) - UI/UX リブランディング デザイン発注書

## 1. プロジェクト概要
グローバルノマド向けメニュー翻訳＆ビジュアライズアプリの名称を「menumenu」から**「ノマメシ（Nomameshi）」**へリブランディングします。
NHKの名物番組「サラメシ（サラリーマンの昼食）」のオマージュであり、世界中を移動しながら働く**グローバルワーカーの「毎日のご飯（メシ）」**を支える実用的なツールとしての立ち位置を明確にします。

## 2. ターゲットペルソナ
- **ワークスタイル**: スローノマド（数日単位のホッピングではなく、1箇所に腰を据えて「暮らすように働く」スタイル）。
- **食の傾向**: 
  - フーディ（食への関心が高い）、雑食。
  - 観光客向けの店やリゾート食よりも、地元の人が日常的に通う「伝統料理・ローカルフード」を好む。
  - お酒も楽しむ（同時に、気分に合わせてソバーキュリアスな選択も尊重する）。
- **コミュニケーション**: 
  - 現地の食堂のスタッフと「美味しいの様々な表現」で会話したり、ローカルな繋がりを持つことを楽しむ。

## 3. デザインの方向性（Tone & Manner）
- **目指す印象**: 「日常・手触り感・ローカル・温かみ・実用性」
- **現在の課題**: 現状の「ダークでソリッド」なUIは、少しサイバー感やテクノロジー感が強すぎます。
- **避けるべき要素**: 「リゾート感、バカンス感、旅行者っぽさ」は不要です。あくまで「仕事の合間の日常のメシ」を彩るUIを目指します。
- **ヒント・要望**:
  - テクノロジーの冷たさを和らげ、ローカル食堂のメニュー表や、ノマドの落ち着いたワークスペースに馴染むような、温かみのあるトーン（アースカラー、ウォームグレー、クラフト感など）を検討してください。
  - 背景や装飾は主張しすぎず、主役である**「解析された料理の写真（背景のHero画像）」**を最も美味しく見せるノイズレスなキャンバスであるべきです。

## 4. UIの前提構造（制約事項）
1. **背景レイヤーとグラスモーフィズム**
   - 料理の全体像（テーブルスプレッド画像）は、画面全体の**「奥（最背面レイヤー）」**に縦長（9:16比率など）で表示されます。
   - 手前に表示される各メニュー項目のカードは、この画像の上に**「半透明ですりガラスのように透ける（backdrop-filterを用いたグラスモーフィズム）」**状態でオーバーレイ配置されます。
   - このため、**文字の視認性（コントラスト）を損なわずに、背景のシズル感をうっすらと感じさせるカードデザイン**を設計してください。
2. **待機画面の「Tips」コーナー**
   - 画像解析中に数秒のローディングが発生します。この時間を利用して「現地のローカル食習慣」や「ノマド事情」に関するTips（豆知識）を表示するセクションがあります。ここも退屈させないデザイン・アニメーションなどをご検討ください。

## 5. デザイナーへの依頼事項
以下の設計・提案をお願いします。
1. **「ノマメシ」のロゴ（タイポグラフィ中心のミニマルなもの等）**
2. **メインカラー / サブカラーパレットの再定義**
3. **タイポグラフィ（フォント）の選定**（視認性が高く、少し親しみのあるもの）
4. **主要画面（トップ画面待機中Tips部、解析結果のグラスモーフィズム画面）のカラースキームを反映したモックアップ作成**
