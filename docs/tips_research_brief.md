# ノマメシ (Nomameshi) - ローカルTips リサーチ発注書

## 1. 概要
グローバルノマド向けメニュー解析アプリ「ノマメシ」において、画像の解析待機中（数秒〜十数秒）に表示する「ローカルTips」の作成をお願いします。

### 対応リージョン & 現状Tips数
| Region | Code | 日本語 | English | ステータス |
|--------|------|--------|---------|-----------|
| スペイン | ES | 22 | 22 | **完了** |
| 日本 | JP | 3 | 3 | 要拡充 |
| イタリア | IT | 3 | 3 | 要拡充 |
| フランス | FR | 3 | 3 | 要拡充 |
| アメリカ | US | 3 | 3 | 要拡充 |
| 韓国 | KR | 3 | 3 | 要拡充 |
| タイ | TH | 3 | 3 | 要拡充 |
| 台湾 | TW | 3 | 3 | 要拡充 |
| デフォルト | default | 3 | 3 | 要拡充 |

### 対応言語
**日本語・英語の2言語のみ**（他言語のTipsは不要）。

## 2. ターゲットペルソナ
- **スローノマド**: 1カ所に数週間〜数ヶ月滞在し、現地の日常に溶け込むように働く層（短期旅行者ではない）。
- **フーディ・雑食**: 食への探求心が強い。観光地の定番より、地元の人が食べる日替わり定食や伝統料理を選ぶ。
- **酒飲み＆ソバーキュリアス**: ローカルのお酒も楽しむが、飲まない日の選択肢も知っておきたい。
- **ローカルコミュニケーター**: お店の人とのちょっとした会話（現地の言葉での「美味しい」の伝え方など）を楽しみたい。

## 3. Tipsの要件・トーン
単なる「旅行ガイドブックの抜粋情報」ではなく、**「現地に住むノマドワーカーの視点」**や**「食文化のリアル・裏技」**をミックスした内容にしてください。

- **文字数**: 1Tipsにつき、1〜2文程度（スマホ画面でサッと読める長さ）。
- **トーン**: 押し付けがましくなく、現地に詳しい友人が教えてくれるような親しみやすいトーン。
- **カテゴリ**:
  1. **ローカル食習慣 & ノマド事情** `#habits`: 食事の時間帯とワークスタイルの兼ね合い、カフェ事情など。
  2. **伝統料理・定番のリアル** `#food`: 「とりあえずこれ頼んどけ」的な知識や、地元民ならではの食べ方。
  3. **地域スタイル解説** `#regional`: メニューに出てくる「〜風」の意味を解説。
  4. **コミュニケーション** `#comms`: スタッフとの距離が縮まる、気の利いた現地の言葉使い。
  5. **お酒・ドリンク事情** `#drinks`: お酒の定番、またはノンアルコール（ソバーキュリアス向け）の気の利いた選択肢。

## 4. 求めるアウトプット
各リージョンにつき **15〜20個** のTipsを **日本語・英語の2言語** で作成してください。

### フォーマット
スペイン編（docs/tips_es.md）に倣い、以下の形式でまとめてください:
```
# Nomameshi Local Tips【[国名] / [CODE]】
> tags: #habits #food #comms #drinks #regional | region: [CODE]

## 🕐 ローカル食習慣 & ノマド事情
1. `#habits`
[日本語Tips]

## 🍽️ 伝統料理・定番のリアル
6. `#food`
[日本語Tips]

（中略）

---
（英語版を同様のセクション構成で続ける）
```

### プレフィックス規約
- 日本語: `💡 豆知識: ` で始める
- 英語: `💡 Tip: ` で始める

### 作成例（スペイン編より）
- **ローカル＆ノマド**: 「💡 豆知識: スペインの朝食は2回。7時にコーヒーだけ、10〜11時にボカディージョで本命の一食。この時間帯がカフェで一番空いてWi-Fiも快適。」
- **コミュニケーション**: 「💡 豆知識: 「美味しい！」は ¡Riquísimo!（リキシモ！）で2割増し伝わる。BuenoやDeliciosoより感情が乗って、店のおばちゃんの反応が段違い。」
- **ドリンク**: 「💡 豆知識: 最初の一杯は Una caña（ウナ・カーニャ）。小さめの生ビールで€1〜2。カーニャをおかわりしながらバルをはしごするのがローカル流。」
- **食のリアル**: 「💡 豆知識: パエリアは地中海沿岸生まれの「昼メシ」。夜メニューにあるのはほぼ観光客向け。ランチ限定の店で食べるのが正解。」

## 5. 実装上の注意
- Tips はアプリの `CULTURAL_TIPS` オブジェクト（`src/app/page.tsx`）に直接埋め込む。
- 各リージョンは `"Japanese"` と `"English"` の2キーのみ。
- 他言語（Chinese, Korean, Spanish, French）のTipsが既存リージョンに残っている場合、新Tips追加時に削除してよい。
- Tips表示は Fisher-Yates シャッフルでランダム化、3.5秒ごとにローテーション。
