# Nomameshi (メニュー翻訳＆視覚化アプリ)

**「メニューを読むのではなく、料理の魅力を視覚的に味わう」** をコンセプトにした、AI駆動のメニュー翻訳・解析・視覚化アプリです。
外国語のメニュー画像をアップロード・ペーストするだけで、単なる直訳ではなく、料理の構成やレストランの雰囲気を解析し、美しいUIで多言語化して表示します。

## 主要機能
1. **メニューの解析・翻訳 (Gemini API利用)**
   - 画像（写真/スクリーンショット）からのOCR＆構造化
   - 対応言語: 日本語、英語、中国語、韓国語、スペイン語、フランス語
   - 単なる翻訳だけでなく、「レストランの雰囲気 (Vibe)」「料理の説明 (Description)」「価格」まで抽出
2. **テーブルイメージ（Hero画像）の自動生成**
   - 抽出した料理群（Image Query）をもとに、そのメニューの料理が並んだ「テーブルスプレッド画像」を生成
3. **共有・保存機能**
   - 翻訳結果テキストのクリップボードコピー / ネイティブShare機能
   - 翻訳結果全体を美しいPNG画像として保存

## 開発ノート / Constraints
- **Geminiモデルの選定について（超重要）**
  - 現在、画像からメニュー項目を構造化・翻訳する推論エンジンには **`gemini-2.5-flash-lite`** を採用しています。
  - 当初は `gemini-2.0-flash` を採用していましたが、テストの結果、`2.5-flash-lite` の方が応答速度が約20%高速（12秒 → 9秒台）であり、精度もメニュー翻訳用途では十分であることが実証されました。
  - **※今後の変更時注意※**: ユーザー体験において「待ち時間の短縮」が最重要指標であるため、無闇に `gemini-2.0-flash` 等の重いモデルに**戻さない**でください。最新の軽量モデル（Flash-Lite系）を常に優先して検証・採用することを推奨します。
